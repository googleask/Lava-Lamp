Lava lamp navigation

Create a drop down menu with a lava lamp using Jquery

Download this project as a .zip file Download this project as a tar.gz file

Introduction

This project will help you how to create a lava lamp navigation menu using Jquery.This is not a Jquery plugin,however its ease of use is impressive.

Usage

HTML

To use it use need to create a markup first,for this make a div with a class='nav'. <div class='nav'> .... </div> Under this you just have to follow a traditional markup of navigations that is using ul and li.For child elements,add them under the li of a parent as I done below:

<ul>
<li><a href='#'>Home</a></li>
<li><a href='#'>Parent</a>
<li><a href='#'>Child 1</a></li>
<li><a href='#'>Child 2</a></li>
<li><a href='#'>Child 3</a></li>
</li>
</ul>

CSS

The CSS is quite simple to handle all you need to do is to just copy the whole CSS and paste in your page.If you want to change colour of the lamp then you need to do a few changes.

First do change background-color of .lamp span

.lamp span {
    display:block;
    height:4px;
    background:#ee6666;
    position: relative;
}

Secondly change the border-bottom-color in the below part.

.lamp span:after {
    bottom: 100%;
    left: 50%;
    border: solid transparent;
    content:" ";
    height: 0;
    width: 0;
    position: absolute;
    pointer-events: none;
    border-color: rgba(238, 102, 102, 0);
    border-bottom-color: #ee6666;
    border-width: 4px;
    margin-left: -4px;
}

Jquery

Their is no need of changing the Jquery,but a point need to be discussed here.

If their is a link that is currently open in navigation menu then it should be classed as selected because according to script,the lamp will stay under the element having a class selected.If their is no element having class selected then lamp will always stay at the first element,that is generally home link.
